Please use a code block!

Why?

Pasting raw code into a chat or forum hurts readability and makes it harder for people to help you. The reasons for this are listed below:

  • It's hard to read.

    Computer code is always rendered using a monospace typeface. This means each character is a set width. W is the same width as I for example. When programmers see code that's not rendered with a monospace typeface, such as in a chat app or internet forum, it's not how they're used to reading code and makes it a little harder to read quickly.

  • It blends with surrounding text.

    Code and discussion should be visually separate. If you're describing a problem in a piece of code then write the code directly underneath without using code blocks, it can be difficult to distinguish where the explanation ends and the code begins.

  • Some platforms support syntax highlighting!

    Most platforms that use Markdown code blocks also support syntax highlighting for common programming languages and file formats! By using a code block with highlighting, it makes the code more readable by programmers who are accustomed to reading code in their editors.

How do I post code properly?

Most platforms have settled on Markdown as the preferred formatting language: 

```js
(async function(){
  let books = await getAllBooks();
  console.log(books);
})();
```

Notice how js follows the ```, this indicates which syntax highlighting to use.

Hopefully this has been informative. Now whenever you're helping someone and they share some code or error messages directly into the chat without using a code block, help them anyway but direct them here so they don't make the same mistake next time!
